Research

Research Questions

This project investigates three core questions at the intersection of NLP and environmental journalism.

RQ1 — Classification Accuracy

How accurately can zero-shot NLI models (DeBERTa) classify sustainability news into five environmental themes without any labelled training data?

RQ2 — RAG Summary Quality

Does retrieval-augmented generation produce summaries that are more relevant and better grounded than generic prompting, as measured by G-Eval?

RQ3 — Baseline Comparison

How do keyword baselines, GPT zero-shot, and DeBERTa NLI compare on macro F1 across the five sustainability categories?

Methodology

Technical Methods

A reproducible pipeline combining news retrieval, LLM generation, neural classification, and automated evaluation.

Data

GDELT 2.0 API

The Global Database of Events, Language and Tone indexes millions of news articles daily. We query the Document API with theme-specific keywords, filtering for English-language sources.

Generation

RAG with GPT-4o-mini

Retrieved article titles and snippets serve as context for GPT-4o-mini, which synthesises a structured 3–4 paragraph summary grounded in the retrieved evidence.

Classification

DeBERTa Zero-Shot NLI

We use cross-encoder/nli-deberta-v3-small as a zero-shot classifier. The model scores each article against each theme label via natural language inference, requiring no labelled training data.

Evaluation

G-Eval Framework

Following Liu et al. 2023, we use GPT-4o-mini as an automated judge to score summaries on relevance (alignment with source articles), coherence (logical flow), and grounding (factual faithfulness).
